## Setup Instructions

1. Install dependencies by running:
`python setup.py`

2. stuff about the local models

## Running the model

Run using:

python -m src your/image/path --model modelname

Available models:

- qwen (default)
- anthropic
- chatgpt

Example:
python -m src images/meme.jpg --model chatgpt
